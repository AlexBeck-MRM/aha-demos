# Brand Guidelines

## Purpose
Hold the closest thing to official AHA brand source material, plus any validated proxy signals already present in the repo.

## Current grounded files
- `assets/aha-logo-red.png`
- `assets/aha-logo-white.png`
- `slides/tokens-v1.6.json`
- `slides/tokens-studio-v1.6.json`
- `slides/aha-slide-system-v1.6.md`

## Signals already visible in the repo
- AHA red should remain a controlled signature accent, not a constant highlight color.
- Bright white surfaces and restrained neutrals create the best foundation for authority and usability.
- Token discipline already exists in the slide system and should inform digital-system thinking.
- Layout and typography decisions benefit from explicit contracts rather than one-off art direction.

## Missing official inputs to ingest next
- Published brand guidelines PDF or deck.
- Approved typography guidance for digital.
- Approved photography and illustration guidance.
- Any accessibility or legal brand constraints that must remain fixed.

## Retrieval note
Use this folder before inferring brand behavior from experimentation alone.
When the official source is missing, say that the guidance is a working direction, not a confirmed brand fact.
