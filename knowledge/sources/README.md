# Source Library

This folder is the first retrieval layer for the AHA design brain.
Use it to point Codex at real source material before relying on distilled summaries.

Current source clusters:
- `aha-brand-history/` for historical brand research, logo evolution, archive captures, red-symbol logic, and agency/in-house stewardship notes.
- `discovery-playback/` for workflow notes, research playback, and system-thinking inputs.
- `brand-guidelines/` for brand signals, logo assets, and visual guardrails.
- `current-site-audit/` for current-state evidence, competitor-style captures, and ecosystem scans.
- `figma-design-workbench/` for live board structure, section references, and board-native annotations.
- `experience-insights-mergers/` for merged experience research, especially the reusable hypotheses and provisional conclusions behind content, navigation, engagement, editorial, and homepage decisions.
- `stakeholder-notes/` for named stakeholders, unresolved questions, and process assumptions surfaced so far.
