# Codex Critique Prompt

## Goal
Critique a proposed direction using the repo's current source hierarchy and prior decisions.

## Required input blocks
- `Question`
- `Relevant sources`
- `Current decision state`
- `What needs critique`

## Output contract
- Sources reviewed with exact file paths
- Decision IDs consulted
- Known ambiguities or contradictions
- What is working
- What is weak or generic
- What conflicts with prior decisions
- What to change next
- Which files should be updated
