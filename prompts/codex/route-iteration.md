# Codex Route Iteration Prompt

## Goal
Push Route A and Route B forward without collapsing them into one direction.

## Prompt blocks
1. `Shared structural foundation`
2. `Route to evolve`
3. `What must stay fixed`
4. `What feels unresolved`
5. `Requested output`

## Output contract
- Sources reviewed with exact file paths
- Decision IDs consulted
- Known ambiguities
- Direction
- Reasoning
- Differences from the other route
- Risks
- File updates to propose
