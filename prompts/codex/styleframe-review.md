# Codex Styleframe Review Prompt

## Goal
Review a styleframe as an art director and system thinker at the same time.

## Prompt blocks
- What the frame is trying to prove
- Which modules are present
- Which route it belongs to
- What feels strong
- What feels generic or unresolved

## Output contract
- Sources reviewed with exact file paths
- Decision IDs consulted
- Known ambiguities
- Overall read
- What the frame proves
- What it still fails to prove
- Priority changes
- Relevant log or inventory updates
