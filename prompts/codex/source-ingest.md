# Codex Source Ingest Prompt

## Goal
Turn new raw material into grounded repo knowledge without skipping source fidelity.

## Required input blocks
- `New source files`
- `Why they matter`
- `Existing files that may be affected`

## Output contract
- source facts extracted
- unresolved ambiguities
- distilled files to update
- logs to update
- any conflicts with current repo guidance
