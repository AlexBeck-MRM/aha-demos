# AHA Flat Clinical Line Icon Sheet Prompt

## Use
Use this prompt to generate a bitmap reference sheet for a flat, line-based AHA icon style candidate.

This is a raster style reference, not a vector export and not the accepted icon direction. Keep it useful for reviewing whether a simpler line system can sit beside or replace the current spatial icon style later.

## Direction
Use one icon language only: `Flat Clinical Line`.

The icons should feel clean, adult, precise, warm, and website-ready. They should be flat line icons with consistent stroke weight, rounded caps, restrained detail, and a small amount of AHA red used only for meaning.

## Prompt
Create a single bitmap raster reference sheet for a new American Heart Association website icon style.

Canvas:
Square image. Pure white `#ffffff` background. Arrange exactly 12 icons in a strict 4-column by 3-row grid. Every icon must sit centered in its own invisible square container with equal scale, even spacing, generous gutters, and no visible boxes. No labels, headings, numbers, UI chrome, logos, watermark, or text.

Style:
Use one style only: `Flat Clinical Line`. This is a bitmap rendering of flat line icons, not a vector file. Icons should look clean, adult, precise, warm, and website-ready. Use simple monoline strokes with rounded caps and rounded joins. Use mostly open linework, not filled illustration. Keep icons flat: no 3D, dimensional shading, gradients, shadows, bevels, texture, perspective, or product-render look.

Line behavior:
Use a consistent medium stroke weight across the whole sheet. Use smooth simplified geometry. Preserve enough internal detail to identify each subject, but keep each icon readable at small UI sizes. Line endings should feel polished and intentional. Use no more than one or two small AHA-red signal strokes per icon.

Colour:
Use dark warm charcoal linework, approximately `#2f3437`. Use AHA red, approximately `#c8102e`, only as a purposeful signal: pulse line, alert mark, AED mark, quit slash, heart cue, sample liquid line, reading marker, or support cue. Optional very pale warm grey line segments are allowed only for secondary structure. The style should read as line-first and flat. No multicolor palette, filled color blocks, blue healthcare palette, or green wellness palette.

Subjects, exactly this order left to right, top to bottom:
1. Blood pressure: cuff, bulb, tube, small gauge with one red reading marker.
2. Heart rhythm: heart outline with one red rhythm line integrated through it.
3. CPR / AED: simplified torso or pad and hands pressing, plus compact AED outline with small red lightning or heart mark; no realistic person.
4. Acute warning signs: rounded square alert tile outline with red exclamation mark.
5. Healthy eating: bowl outline with simple apple and leaf forms, one red apple or produce signal.
6. Movement: simple shoe outline with minimal sole detail.
7. Sleep: bed outline with pillow and blanket lines, no moon or window.
8. Mental wellbeing: head profile outline with small red heart or pulse cue inside, no brain detail.
9. Quit smoking / vaping: cigarette or vape outline with one red diagonal slash, no large prohibition ring.
10. Volunteer / community: two abstract support hands or linked forms holding one small red heart, no detailed fingers.
11. Research / science: microscope outline and simple flask line with red liquid line.
12. Donate / support: card or coin outline with one small red heart mark, no currency symbol.

Composition:
Each icon should occupy about `60-70%` of its invisible cell. Keep consistent optical weight. Use centered, calm, clear silhouettes. Avoid cropped icons.

Avoid:
Vector-file appearance with anchor points, labels, text, numbers, letters, UI annotations, boxes, containers, logos, watermarks, filled illustration, filled color blocks, 3D, shading, shadows, gradients, bevels, texture, realistic rendering, product rendering, dense detail, tiny gauge ticks, small labels, individual fingers, faces, hair, skin tones, clinician-patient scenes, decorative dots, sparkles, background shapes, blobs, backplates, halos, multicolor accents, stock clipart feel, childish proportions, emoji-like hearts, and cute mascot energy.

## Quality Check
- Exactly 12 icons appear in a 4 x 3 grid.
- The file is a bitmap reference image, not an SVG or source vector.
- The style reads as flat line iconography.
- Stroke weight and optical scale are consistent across the sheet.
- Red appears only as a small meaning signal.
- There are no labels, text, numbers, boxes, or UI annotations.
- There are no shadows, gradients, dimensional effects, or filled illustration blocks.

## V4 Duo-Colour Iteration Prompt
Use this prompt when the flat line style needs to keep the chunky no-background direction while proving the icons can work with a second colour.

Create a fourth bitmap raster reference sheet for the American Heart Association flat-line icon style candidate.

Design target:
Use one style only: `Chunky Duo Line`. Keep the chunky red outline attitude, rounded geometry, simplified AHA infographic feel, and no background circles from the latest version. Bring back a second colour as a real supporting stroke system, not just tiny accents. The result should prove the icons can work in two colours without becoming busy.

Canvas:
Square image. Pure white `#ffffff` background. Arrange exactly 12 icons in a strict 4-column by 3-row grid. Every icon must sit centered in its own invisible square container with equal scale, even spacing, generous gutters, and no visible boxes. No circular backgrounds, blush fields, labels, headings, numbers, UI chrome, logos, watermark, or text.

Style:
Flat bitmap icon reference sheet. Thick rounded monoline strokes, rounded caps, rounded joins, simplified silhouettes, and minimal internal detail. No 3D, dimensional shading, gradients, shadows, bevels, texture, perspective, or product-render look. The icons should feel chunky, friendly, clean, and mature enough for a health nonprofit website.

Two-colour system:
Use AHA red, approximately `#c8102e`, as the primary stroke colour and deep AHA-adjacent purple, approximately `#4b1b75`, as the secondary stroke colour. Use both colours across the sheet. Red should usually carry the main recognisable AHA, heart, action, or attention signal. Purple should carry secondary structure: supporting object edges, interior utility details, secondary icon parts, or alternate emphasis. Do not use black, charcoal, extra colours, pale circles, or any background colour.

Colour balance:
Aim for roughly `60%` red linework and `40%` purple linework across the sheet. Each icon should use both colours where it makes sense, but keep the logic clean. Avoid random colour switching inside one continuous shape. Red should feel intentional, not decorative. Purple should make the icon easier to read, not busier.

Subjects, exactly this order left to right, top to bottom:
1. Blood pressure: cuff, bulb, tube, and small gauge. Use purple for the cuff/tube support structure and red for the gauge marker or key signal.
2. Heart rhythm: red heart outline and rhythm line, with a small purple support segment if needed.
3. CPR / AED: simplified chest pad/torso plus pressing hands and compact AED mark. Use purple for body/support structure and red for AED/heart/lightning signal.
4. Acute warning signs: rounded square alert tile. Use purple for the tile outline and red for the exclamation mark.
5. Healthy eating: apple or bowl with simple leaf. Use red for apple/primary food shape and purple for leaf/bowl/support line.
6. Movement: walking shoe. Use red for main shoe silhouette and purple for sole or two large detail strokes.
7. Sleep: bed with pillow and blanket. Use purple for bed frame/support line and red for blanket or rest cue.
8. Mental wellbeing: simplified seated meditation/stress icon or head profile with heart cue. Use purple for body/head structure and red for heart/calm cue.
9. Quit smoking / vaping: cigarette/vape line with diagonal slash. Use purple for cigarette/vape object and red for the slash signal.
10. Volunteer / community: simplified group/support icon or hands holding a heart. Use purple for people/hands/support structure and red for heart/action signal.
11. Research / science: simplified microscope. Use purple for microscope structure and red for lens/sample/action cue.
12. Donate / support: clipboard/card with heart mark. Use purple for card/clipboard outline and red for heart signal.

## V5 Warm Grey Recolour Rule
Use this rule when the chunky duo-line style needs a calmer second colour.

Start from `aha-flat-clinical-line-12-sheet-v4-duo-colour.png`.

Keep:
- chunky rounded bitmap linework
- pure white sheet background
- AHA red signal strokes
- the same 12-subject grid and icon layout

Change:
- replace all purple support strokes with warm grey
- target warm grey: approximately `#7e776f`
- keep antialiasing soft at stroke edges
- do not change red strokes
- do not add background circles, fills, shadows, gradients, or extra colours

Result:
The icon system should read as red plus warm grey, where red carries the AHA/action signal and warm grey carries secondary structure.
