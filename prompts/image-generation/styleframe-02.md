# Image Prompt Template: Styleframe 02

## Subject block
Design a content-rich AHA website ecosystem frame showing hero, mega menu, tabs or topic band, cards, article top, contribution module, search or discovery, and a community moment.

## Expression block
Keep the system bright, warm, trustworthy, tactile, and clearly branded while proving it can scale beyond a single hero.

## Composition block
Show useful hierarchy across varied modules. The frame should feel operational but not sterile.

## Constraints
- Avoid dashboard stiffness.
- Avoid excessive red micro-highlighting.
- Avoid generic stock-photo collage energy.
