# AHA Spatial Icon Style Exploration Prompt

## Use
Use this prompt to generate a raster comparison sheet for three possible spatial evolutions of the AHA restrained-solid icon language. This is a style exploration, not production icon export.

## Design rules
- Keep the icons mature, useful, and system-ready. Avoid cute toy objects, emoji hearts, plush shapes, mascot energy, and app-store icon gloss.
- Keep the white-first AHA website feel. Spatial depth should add tactility and ownership, not visual weight.
- Use light website greys as the foundation: white, off-white, cool bone, light stone grey, pale warm grey, muted taupe-grey, and soft charcoal.
- Greys should feel warm but not beige, tan, brown, clay, sand, oat, sepia, or muddy.
- Use colour selectively. Red is a signal, not the default fill. Non-red colour is allowed when it clarifies subject matter.
- Keep red separated from warm or darker neutrals through pale buffers, white cuts, negative space, or clear value contrast.
- Use one consistent light source across the sheet, preferably upper left.
- Use shallow depth only. Avoid glossy 3D, toy-like bevels, inflated clay shapes, dramatic shadows, or realistic render material.
- Sleep must use visible contrast: neutral base, medium neutral main form, and only a small water/air tint.
- One route should test an Airbnb-like `Lava Object` approach: iconic, metaphor-led, softly sculpted objects with a little retro warmth. Do not copy Airbnb brand assets, exact icons, colours, logo shapes, or campaign compositions.

## Three styles
1. `Low Relief`
   - Almost flat, with raised layers and very shallow contact shadows.
   - Front-facing or near-front-facing.
   - Feels like layered matte paper, not illustration stickers.
   - Best if we want the current flat system to become just a little more tactile.

2. `Lava Object`
   - Airbnb-adjacent in spirit: iconic, metaphor-led, dimensional objects that feel crafted and memorable.
   - Think softly moulded forms, softened edges, subtle retro warmth, and simple object metaphors.
   - More sculptural than Low Relief, but less technical and less literal than isometric product render.
   - Best if we want the system to feel more ownable, tactile, and emotionally memorable.
   - Keep it AHA-safe: mature, restrained, white-first, no Airbnb logo, no Airbnb colours, no direct copy of Airbnb icons.

3. `Inset Surface`
   - Icons are built as raised and recessed forms on very pale invisible or near-white surfaces.
   - Uses cut-ins, carved negative space, and quiet shadow to show depth.
   - More ownable and tactile, but still restrained.
   - Best if we want a distinctive system language without going full 3D.

## Prompt
Create a single raster image reference sheet for a health nonprofit spatial icon system.

Canvas:
Square image. Pure white #ffffff background. Create a 3-column by 6-row comparison grid. Each column is one spatial style. Each row repeats the same subject across the three styles. Every icon must sit inside its own invisible square container with equal size, even spacing, and generous gutters. Do not draw containers, borders, labels, headings, numbers, logos, UI chrome, watermark, or visible text.

Column structure:
Column 1: Low Relief.
Column 2: Lava Object.
Column 3: Inset Surface.

Row subjects:
Row 1: Blood pressure cuff or gauge.
Row 2: Heart rhythm pulse/heart.
Row 3: CPR / AED rescue cue.
Row 4: Healthy eating bowl/plate/apple.
Row 5: Sleep crescent/pillow/bed with clear neutral contrast.
Row 6: Research / science microscope or flask.

Shared style:
The icons should feel modern, calm, mature, useful, trustworthy, and easy to scan on white website cards and navigation surfaces. Keep forms broad and simple. Use one main subject, one information/action signal, and zero or one support shape per icon. Use a consistent upper-left light source, subtle depth, and restrained shadows. No faces, no full scenes, no decorative props.

Palette:
Use white #ffffff, off-white #f4f3f3, soft white #f5f4f2, cool bone #eeece9, light stone grey #e1e0dc, pale warm grey #d8d9d6, muted taupe grey #7d7971, and soft charcoal #3d3c3a. Use red #e6001b and deep red #bf0000 only as small action, urgency, heart, selection, pulse, AED, gauge, or alert signals. Use secondary colour sparingly and only when tied to the subject: nature green #e7f0eb, water/air blue #edf2f3, soft red support #f6e8e9, rare food warmth #f5ebdd. Avoid black, broad brown, muddy taupe, saturated green, saturated blue, beige dominance.

Colour behavior:
Red should usually sit on white, pale grey, off-white, a clean separated shape, or enough negative space. Do not place red directly inside muddy brown, dark taupe, or overly warm neutral areas. If red overlaps a neutral, create a pale buffer, white cut, clear cutout, or strong value separation. Avoid repeating large red hearts; for heart rhythm, prefer a red pulse line on a neutral heart.

Spatial behavior:
Use subtle dimensionality only: small side planes, slight overlaps, shallow relief, soft contact shadows, and clean cut-ins. Keep the icons lighter than product renders and less playful than toy objects. No glossy gradients, metallic shine, glass, clay render, plastic toy finish, heavy bevels, dramatic shadows, or photorealistic lighting.

Lava Object behavior:
For the middle column only, make the forms feel like simplified, softly sculpted physical objects: rounded but not puffy, dimensional but not glossy, memorable but not childish. Use clean object metaphors and a little retro warmth in the form language, while keeping the AHA palette and white-first restraint. Avoid direct Airbnb references, travel motifs, hotel bells, balloons, houses, or any visual that reads as Airbnb-owned.

Sleep rule:
The sleep row must not be all blue. Use a pale neutral base, a medium neutral or taupe main object, and a small water/air support tint. It should feel adult and functional, not dreamy, cute, plush, or nursery-like.

Avoid:
Cute icons, emoji-like hearts, large repeated red hearts, plush objects, toy-like 3D, app-icon gloss, clay renders, heavy bevels, dramatic shadows, labels, text, numbers, logos, UI screenshots, black strokes, harsh contrast, muddy warm neutrals, red buried in brown, saturated multicolour accents, blue healthcare palette, green wellness palette, beige dominance, tiny decorative details, full illustrated scenes, mascots, childish proportions, expressive faces, stock icon-library sameness, and dense medical diagrams.

## Quality check
- Exactly three spatial styles appear as columns.
- Exactly six subject rows appear, repeated across all three styles.
- The middle column reads as a Lava Object route: crafted, metaphor-led, and gently dimensional without copying Airbnb.
- The result reads more spatial than flat icons but still mature and website-system appropriate.
- Sleep has meaningful contrast and is not one pale-blue object.
- Red is separated, selective, and not repeated as a large heart motif.
