# Icons

## Direction
Build icons from branded contour logic, then simplify them until they hold at small sizes. Keep the set slightly chunky, low-detail, and calm, with hover expressed through very pale fill inside the same drawings rather than a different illustration style.

## Reasoning
- Clear silhouettes help icons work as navigation anchors in dense interfaces.
- Simpler forms feel more ownable and less like a borrowed symbol library.
- Controlled softness keeps the set approachable without losing precision.
- Hover should add warmth and state change without changing recognition or adding noise.

## Current board study
- Line is now the base drawing language.
- Hover is converging toward the same line drawings with very pale duo fills and no red.
- Preferred traits are: fewer lines, smaller icons, stronger silhouettes, and slightly chunkier stroke weight.

## Open questions
- Which icon subjects can lose one more layer of detail without hurting recognition?
- Which existing regions should fill on hover so the activation state stays quiet and consistent across the set?
