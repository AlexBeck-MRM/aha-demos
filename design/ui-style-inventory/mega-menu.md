# Mega Menu

## Direction
Treat the mega menu as a guided orientation surface, not a dense sitemap dump. Use generous spacing, clear grouping, and one featured content or action area.

## Reasoning
The ecosystem is broad, so the menu needs to help people choose a pathway quickly. A calmer, curated mega menu makes utility feel more trustworthy.

## Open questions
Should the featured area lean more editorial or more task-oriented?
