# Grid

## Direction
Use a flexible grid with strong reading lanes, generous spacing, and a few controlled breakouts. The layout should feel open and modern, but the underlying structure needs to stay disciplined enough to support dense health content.

## Reasoning
- Strong lanes help dense modules and long-form content feel easier to scan.
- Controlled breakouts create visual interest without turning the page into a collage.
- The grid is what lets route expression vary while the system still feels related.
- We still need to test how much asymmetry the layouts can absorb before clarity starts to drop.

## Open questions
How often should imagery or illustration deliberately cross grid boundaries?
