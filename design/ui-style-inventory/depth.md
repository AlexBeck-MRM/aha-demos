# Depth

## Direction
Use soft elevation, subtle shadowing, and occasional layered overlap to create tactility. Depth should be visible enough to invite interaction but never heavy-handed.

## Reasoning
Depth is a core part of moving the site away from flat nonprofit-web conventions. It makes buttons, cards, and tabs feel touchable and intentional.

## Guardrail
Avoid dark, muddy shadows or exaggerated glassmorphism.
