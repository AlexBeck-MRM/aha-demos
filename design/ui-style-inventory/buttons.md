# Buttons

## Direction
Buttons should feel tactile, rounded, and slightly organic. Keep the hierarchy simple: primary uses AHA red for the main action, while secondary stays monochrome but still feels spatial and intentional through subtle depth.

## Reasoning
- A simple primary-versus-secondary split keeps calls to action legible across donation, sign-up, and utility flows.
- Secondary buttons still need enough finish to read as intentional, not disabled or left over.
- Slightly softer, more organic corners help the system feel warmer without becoming playful.
- If every action is loud, the interface loses focus quickly.

## Route emphasis
Route A can allow slightly warmer, softer button presence. Route B can keep the same tactility with a cleaner edge definition.
