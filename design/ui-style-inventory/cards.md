# Cards

## Direction
Cards should work as complete click targets, not small layouts that still need extra link labels to explain themselves. Use them where they help browsing or comparison, and make the headline and summary do the action-setting work.

## Reasoning
- Full-card interaction reduces clutter and makes browse surfaces easier to scan.
- Action-led headlines carry intent better than repeating a generic `Read more` label.
- Light elevation helps cards feel touchable without turning the whole site into stacked tiles.
- Selective use keeps the system edited instead of overly modular.

## Route emphasis
Route A can allow slightly softer image-led cards. Route B can use a tighter component rhythm.
