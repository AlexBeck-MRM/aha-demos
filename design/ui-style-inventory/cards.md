# Cards

## Direction
Cards should feel purposeful, lightly elevated, rounded, and well-spaced. Use cards selectively to organize content, not as the default wrapper for everything.

## Reasoning
Cards help manage content density, but overuse can make the ecosystem feel fragmented. Thoughtful card use supports both tactility and clarity.

## Route emphasis
Route A can allow slightly softer image-led cards. Route B can use a tighter component rhythm.
