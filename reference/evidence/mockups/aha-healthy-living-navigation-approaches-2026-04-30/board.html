Six navigation models for guides with supporting articles

The core guide should stay canonical and chapter-led. Supporting articles, tools, recipes, and infographics need to be findable without becoming the main navigation structure again.

Solve ProblemsSetbacks, barriers, edge cases.

Best for dense guides where users need a stable sense of place.
